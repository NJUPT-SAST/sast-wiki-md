# 这是什么？
欢迎来到南京邮电大学大学生科学技术协会（以下简称 SAST）的信息公示站，在这里您可以寻找到：

- 校科协的介绍
- SAST 成员和嘉宾撰写的技术博客，心得体会和其他文章
- 公开分享的讲义、资料和有价值的链接
- 南邮生活指南

本站的搭建离不开以下工具和开源项目的支持，在此一并鸣谢：
